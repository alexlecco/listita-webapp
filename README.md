# Listita

Una webapp de lista de compras estilo Google Keep.

## Cómo funciona

Escribí tu lista en el área de texto — cada línea se convierte en un ítem con formato `- ítem`. La lista se guarda automáticamente en `localStorage` y persiste entre recargas de página.

**Clave en localStorage:** `shopping-list` → `{ items: string[] }`

## Tecnologías

- React + Vite

## Instalación y uso

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
